Welcome to Anybotics Mission Planner

Anybotics Mission Planner is an Angular application (version 16.1.0) designed to facilitate the creation, deployment, and monitoring of missions for Anybotics robots. Users can create missions specifying the robot's desired destinations and then deploy these missions to observe how the robot completes its tasks.

Getting Started

Follow these steps to set up and run the Anybotics Mission Planner:

  1. Clone the Repository: 
    git clone https://github.com/mihaibrewshare/anybotics-mission-planner
  2. Install Dependencies: 
    npm install
  3. Serve the Application: 
    ng serve

Application Features

Navbar

  • Logo: Clicking on the logo redirects to the Home page.
  • Hamburger Icon: Clicking on the hamburger icon opens a menu to help with navigation.
  • Share Button: Clicking on the share button redirects to the project's GitHub repository.

Home Page

The Home Page contains instructions on how to install and use the application, providing a quick start guide for new users.

Planner Page

The Planner Page allows users to create missions for the robot. Key features include:

  • Mission Creation: Users can create a mission by providing a name and specifying the X and Y coordinates (in a Cartesian system) for the robot's destination.
  • Mission Table: All created missions are displayed in a table format for easy viewing and management.

Player Page

The Player Page is used to deploy and monitor missions created on the Planner page. Features include:

  • Deploy Missions: Users can deploy any of the created missions.
  • Animation Control:
    • Play Button: Starts the animation, showing the path taken by the robot.
    • Stop Button: Stops the animation.

This documentation provides a concise guide to the main features and setup of the Anybotics Mission Planner application. For further details and updates, please refer to the project's GitHub repository.
